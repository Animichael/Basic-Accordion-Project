What is HTML used for?
HTML is an acronym which stands for HyperText Markup Language It is a technology that is used in describing the content of web pages As person would mistakenly say that HTML is a programming language , I stand to correct that impression that HTML is not a programming language but rather a markup language mainly used for describing web pages
